
# **The Odin Project – Recipe Page**

A simple webpage created as part of *The Odin Project* foundations course to demonstrate basic HTML and CSS skills.

## **How It’s Made**

**Tech Used:** HTML, CSS

This project showcases a basic recipe webpage built with semantic HTML elements and styled with CSS.

It includes multiple recipe pages connected using anchor tags, allowing users to navigate between different dishes.

## **Features**

* Clean and simple layout
* Multiple recipe subpages
* Easy navigation using anchor links
* Beginner-friendly HTML structure

## **Purpose**

This project serves as a foundational exercise to build comfort with HTML, CSS, and structuring multi-page websites.

It is part of my learning journey and portfolio as I progress through  *The Odin Project* .
